# ⏰ Relógio em Pop-up

## 📌 Descrição

Aplicação simples e funcional que exibe um relógio digital em formato de pop-up, permitindo ao usuário visualizar as horas em tempo real sem sair da página principal. O projeto foi desenvolvido com foco em interatividade e experiência do usuário.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Funcionalidades

* Exibição de horário em tempo real
* Abertura em formato pop-up
* Atualização automática dos segundos
* Interface simples e intuitiva



## 💡 Observações

Este projeto foi desenvolvido para praticar manipulação de tempo com JavaScript e criação de interações dinâmicas na interface.
